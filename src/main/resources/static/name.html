好名字的一生

刘德华原名叫刘福荣。从其原名可以看出，其名比较难听、不雅、不俗，而且给人感觉也是比较土气。 本想要富贵荣华，可难遂愿，被当时的无线封杀，还遭受黑社会的追砍。而后他花了十万港元，拜访了 一位玄学大师，此名听起来大气，有一定的底气、洪亮又顺口、上声，木命为壁上土命，生于辛丑年， 丁酉月，癸亥日，一生有利的行业为电业工业、影视业和房产土地业、演艺业和舞台都为火的行业，所以 自身的本命比较合。改名为刘德华，最终成就天王的地位，也属香港演艺圈少有的不倒翁。
蔡依林原名叫蔡宜凌。蔡依林本命“蔡宜凌”，针对歌唱行业，重新取了一个雷同的艺名，果然让蔡依林 星运一路长红！蔡依林出道前，在“高人指点”的情况下，以姓名学角度取了一个艺名，针对歌唱事业， 唱片公司在改名时，就选用发音类似的名字，以笔画的不同，针对歌唱事业量身订做了一个艺名。果不其然， 蔡依林如今已是华语乐坛呼风唤雨的小天后了。
